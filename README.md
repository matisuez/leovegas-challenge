### <u>Position</u>: <i>NodeJS Engineer</i>

## Project set up

Install dependencies
````bash
npm install
````
Build project
````bash
npm run build
````

### Execute local environment
1. Run development mode
````bash
npm run dev
````

### Execute test environment

1. Run tests
````bash
npm run test
````
or watch mode
````bash
npm run test:watch
````
